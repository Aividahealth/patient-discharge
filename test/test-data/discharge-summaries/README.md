# Test Discharge Summaries

This directory contains sample discharge summaries for testing the portal functionality.

## Files

### Markdown Files (.md)
- `patient-001-discharge.md` - Cardiac patient (MI with stent placement)
- `patient-002-discharge.md` - OB/GYN patient (normal delivery)
- `patient-003-discharge.md` - Orthopedic patient (hip fracture/replacement)
- `patient-004-discharge.md` - Pediatric patient (pneumonia)
- `patient-005-discharge.pdf` - Complex medical patient (PDF format)

### PDF Files (.pdf)
PDF versions are generated for testing binary file uploads and processing.

## Usage

These files are used by the comprehensive portal test suite to:
1. Test discharge summary upload functionality
2. Test display in clinician portal
3. Test simplification workflow
4. Test expert review process
5. Test patient access to simplified versions
6. Test multi-format support (MD and PDF)

## Patient Information

All patient information in these files is **fictional** and created for testing purposes only.
